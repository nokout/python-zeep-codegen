### Option 3: Build with React

If you have a React project, install RJSF and use it directly:

```bash
npm install @rjsf/core @rjsf/validator-ajv8
```

```jsx
import Form from "@rjsf/core";
import validator from "@rjsf/validator-ajv8";
import schema from "./schema.json";

function MyForm() {
  return (
    <Form
      schema={schema}
      validator={validator}
      onSubmit={({formData}) => console.log(formData)}
    />
  );
}
```

## Customization Tips

### Custom Field Widgets

You can customize how fields are rendered using `uiSchema`:

```javascript
const uiSchema = {
  description: {
    "ui:widget": "textarea",
    "ui:options": {
      rows: 5
    }
  },
  email: {
    "ui:widget": "email"
  },
  birthdate: {
    "ui:widget": "date"
  }
};
```
